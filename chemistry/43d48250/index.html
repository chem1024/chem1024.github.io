氮及其化合物

发表于 2019-01-14 | 评论数： | 阅读次数：
本文字数： 3.8k | 阅读时长 ≈ 3 分钟

\(\ce{N2}\)

氮气分子中的 \(\ce{N#N}​\) 键能较大，氮气分子较稳定，常温下难以发生化学反应，常用于作保护气。液氮汽化时吸收大量热，故可用于做冷冻剂。

氮气与非金属单质反应

哈伯法制氨气：\(\ce{N2 + 3H2 <=>[\text{高温高压}][\text{催化剂}] 2NH3}\) 1

雷雨发庄稼：空气中的氮气和氧气在雷电的作用下转化为一氧化氮，又被空气中的氧气氧化为二氧化氮，二氧化氮和雨水作用合成硝酸，落入土壤转化为硝酸盐，从而被植物吸收，实现自然固氮2

\[ \ce{N2 + O2 ->[放电] 2NO}\\ \ce{2NO + O2 -> 2NO2}\\ \ce{3NO2 + H2O -> 2HNO3 + NO} \]

氮气与金属单质反应

金属镁可以在氮气中燃烧生成氮化镁，氮化镁能与水发生水解反应。 \[ \ce{3Mg + N2 ->[点燃] Mg3N2}\\ \ce{Mg3N2 + 6H2O -> 3Mg(OH)2 + 2NH3 ^} \]

氮的固定

自然固氮：高能固氮（空气中的氮气、氧气雷电作用）、生物固氮（豆科植物的根瘤菌、自生固氮菌体有固氮酶都能将游离态的氮转化为化合态）

人工固氮：哈伯法

氮气的制备

  • 鼓空气通过装有灼热铜粉的硬质玻璃管，除去空气中的氧气，剩余的气体大部分为氮气

  • 加热饱和亚硝酸钠（\(\ce{NaNO2}\)）与氯化铵混合溶液制备氮气

  • 制备氨气，将氨气通过装有灼热的氧化铜粉末的硬质玻璃管，再将未反应的氨气除去，可得氮气

氮氧化物

\(\ce{NO}\)

一氧化氮是一种无色无味，有毒的气体，难溶于水。在空气中容易被氧气氧化，故实验室不能用排空气法收集，而只能用排水法收集一氧化氮。 \[ \ce{2NO + O2 -> NO2} \] 该反应在常温下可以完全进行，温度高于 150℃ 时，二氧化氮开始分解，高于 650℃ 时二氧化氮完全分解为一氧化氮和氧气。实验室一般用金属单质与稀硝酸反应制备一氧化氮。 \[ \ce{3Cu + 8HNO3 -> 3Cu(NO3)2 + 2NO ^ + 4H2O} \] 一氧化氮能与血红蛋白中的亚铁离子结合，使人窒息中毒。由于一氧化氮容易与亚铁离子结合，故实验室可用硫酸亚铁溶液吸收尾气中的一氧化氮。3

\[ \ce{FeSO4 + NO -> Fe(NO)SO4} \]

\(\ce{NO2}\)

\(\underset{红棕色}{\ce{NO2}}\)室温下为红棕色，具有刺激性气味的有毒气体。熔点为 -11.2℃ ，沸点为 21.1 ℃ 。\(\ce{NO2}\) 容易二聚为 \(\ce{N2O4}\)，温度低于 21.1 ℃ 时，完全转化为液态 \(\underset{无色}{\ce{N2O4}}\) ，温度高于 150℃ 时，\(\ce{N2O4}\) 完全解离为 \(\ce{NO2}\)。 \[ \ce{2NO2 <=> N2O4} ;\quad \Delta H < 0 \] 二氧化氮能与水发生歧化反应，故二氧化氮不能用排水法收集，应用向上排空气法。 \[ \ce{3NO2 + H2O -> 2HNO3 + NO} \] 这个反应实际上是分两步进行的，二氧化氮先与水发生歧化反应生成硝酸和亚硝酸，亚硝酸不稳定发生分解。 \[ \ce{2NO2 + H2O -> HNO3 + HNO2}\\ \ce{3HNO2 -> HNO3 + 2NO + H2O} \] 二氧化氮能溶于氢氧化钠溶液中，发生歧化反应，该反应是实验室吸收二氧化氮尾气的常用方法。 \[ \ce{2NO2 + 2NaOH -> NaNO3 + NaNO2 + H2O} \] 虽然一氧化氮不能溶于氢氧化钠溶液中，但是二氧化氮与一氧化氮混合气体却可以溶于氢氧化钠溶液归中生成亚硝酸钠。 \[ \ce{NO + NO2 + 2NaOH -> 2NaNO2 + H2O} \] 二氧化氮具有较强的氧化性，一定条件下可以与一氧化碳、氨气等还原性气体反应。 \[ \ce{2NO2 + 4CO -> N2 + 4CO2}\\ \ce{6NO2 + 8NH3 -> 7N2 + 12H2O} \]

氮氧化物的污染和防护

氮氧化物会对环境造成严重的污染，如光化学烟雾、破坏臭氧层、形成硝酸型酸雨。常用的处理氮氧化物尾气的方法有碱液吸收法和催化转化法。

  • 碱液吸收法：工厂用 \(\ce{Na2CO3}​\) 吸收氮氧化物，并可得到副产品亚硝酸盐。
  • 催化转化法：汽车排放的尾气中含有 \(\ce{NO_x}\) 和 \(\ce{CO}\) ，选用合适的催化剂可将其转化为无毒无害的 \(\ce{N2}\) 和 \(\ce{CO2}\) 。

硝酸

纯硝酸为无色，易挥发的液体，有刺激性气味。浓硝酸见光、受热易分解，产生红棕色的二氧化氮，故浓硝酸呈现黄色，需避光保存。 \[ \ce{4HNO3 -> 4NO2 + O2 + 2H2O} \] 稀硝酸和浓硝酸都具有很强的氧化性，可以和金属发生反应。 \[ \ce{3Cu + 8HNO3(浓) -> 3Cu(NO3)2 +2NO ^ + 4H2O}\\ \ce{Cu + 4HNO3(稀) -> Cu(NO3)2 + 2NO ^ + H2O} \] 铁与稀硝酸发生反应时，由于 \(\ce{Fe}\) 可以还原 \(\ce{Fe^{3+}}\)，\(\ce{HNO3}\) 可以氧化 \(\ce{Fe^{2+}}\) ，所以\(\ce{\frac{n(Fe)}{n(HNO3)}}\) 不同，产物可能不一样。 \[ \ce{3Fe + 8HNO3 -> 3Fe(NO3)2 + 2NO ^ + 4H2O}\\ \ce{Fe + 4HNO3 -> Fe(NO3)3 + NO ^ + 2H2O} \] 值得注意的是，铁、铝遇到冷的浓硝酸或浓硫酸时，会发生钝化，在金属表面形成致密的氧化膜保护金属不被腐蚀。

工业上用氨气的催化氧化来制备硝酸，氨气催化氧化得一氧化氮，一氧化氮又被氧气氧化为二氧化氮，二氧化氮用水吸收得到硝酸。 \[ \ce{4NH3 + 5O2 <=>[\text{催化剂}][\Delta] 4NO + 6H2O}\\ \]

氨气

氨气为无色具有刺激性气味的气体，密度小于空气，极易溶于水。

氨气的水溶液被称之为氨水，呈碱性。 \[ \ce{NH3 + H2O <=> NH3.H2O <=> NH4^+ + OH-} \] 氨气与氯化氢气体在常温下，反应产生白烟。 \[ \ce{NH3(g) + HCl(g) -> NH4Cl(s)} \] 氯化铵固体受热容易发生分解，生成氯化氢和氨气。 \[ \ce{NH4Cl(s) ->[\Delta] NH3(g) + HCl(g)} \] 氨气具有一定的还原性，能被氯气氧化为氮气。生成的产物氯化氢又能继续与氨气反应，故氨气与氯气反应也能观察到有白烟产生。 \[ \ce{2NH3 + 3Cl2 -> N2 + 6HCl} \] 氨气与氧气在不同条件下可以发生不同反应，氨气在纯氧中燃烧生成氮气，也能与氧气发生催化氧化。 \[ \ce{4NH3 + 3O2 -> 2N2 + 6H2O}\\ \ce{4NH3 + 5O2 <=>[\text{催化剂}][\Delta] 4NO + 6H2O} \] 高温下，氨气是强还原剂，能还原某些氧化物。 \[ \ce{3CuO + 2NH3 -> 3Cu + N2 + 3H2O} \]

氨气的实验室制法

  • 氯化铵和氢氧化钙混合，固固加热 \[ \ce{NH4Cl + Ca(OH)2 ->[\Delta] CaCl2 + NH3 ^ + H2O} \]

  • 浓氨水与氢氧化钠固体，固液混合不加热 \[ \ce{NH3 + H2O <=> NH3.H2O <=> NH4^+ + OH-} \] 氢氧化钠固体溶于水放出大量热，氨气溶解度下降，氨气逸出；且溶液氢氧根离子浓度上升，平衡逆向进行，有利于氨气的逸出。

  • 浓氨水与生石灰，固液混合不加热

    氧化钙与水反应放热，氨气溶解度下降，氨气逸出；且溶剂减少，有利于氨气的生成。

铵根离子（\(\ce{NH4+}\)）的检验

向试液中加入==浓==氢氧化钠，==加热==，产生气体若能使==湿润红色==石蕊试纸变蓝，则证明溶液中存在 \(\ce{NH4+}\) 。
